**Sace VR**
A simple VR shooter made with Unity 6 and Google Cardboard XR Pluggin for android

The project contains a small arcade-style VR game where the player looks arround, aims with a center dot, and destroys incoming enemies using the Cardboard trigger input.
Check out the releases section for the apk of feel free to build from source.
