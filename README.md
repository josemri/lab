**projects**

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

A collection of small, self-contained projects exploring different ideas and technologies. Each subdirectory is an independent project with its own scope and toolchain.
<br clear="both"/>

## projects

| Project | Description | Language |
|---------|-------------|----------|
| **cardboard** | VR shooter for Google Cardboard | Unity / C# |
| **craftOS** | CC:Tweaked turtle scripts (mining, scanner) | Lua |
| **cube** | ASCII Rubik's Cube simulator with GiiKER smart cube support | Python |
| **snake** | Terminal-based Snake game | C |
| **vidTUI** | TUI movie browser and player using TMDB & headless Firefox | Rust |
